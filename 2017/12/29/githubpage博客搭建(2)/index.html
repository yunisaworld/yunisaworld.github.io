githubpage博客搭建(2)

发表于 2017-12-29 | 分类于 github page | 浏览 次

上一篇说明了如何搭建博客，这里说明一些需要修饰的东西。

基本信息设置

语言设置

在博客目录的_config.yml中配置language选项，默认是英文

作者

在博客目录的_config.yml中配置author选项

标题

在博客目录的_config.yml中配置title选项

侧边头像

这里需要在主题的_config.yml中设置avatar，这里可以通过两种方式，一种是在主题目录下，一种是在博客目录下，这里选择在博客目录下面设置，因为这里有两种好处，一种是其它主题也可以用这个图像，另外一个是这样可以将图片和md文件一起管理，可以放到git上，避免在theme中不方便管理。

功能设置

分类

博客_config.yml中配置

主题_config.yml中配置

新建

修改index.md

标签

标签和分类类似，只需要把相应的地方写成tags。

添加访问次数

在主题配置文件theme/next/_config.yml中

页面显示

页面底部

打开blog/themes/next/layout/_partials/footer.swig，使用注释，或者修改一下内容。

设置预览

next主题默认是全部显示的，因此需要自己去设置，在主题的_config.yml文件中找到auto_excerpt项，然后将enable设置为true。

图片边框

图片默认有边框，可以通过以下地方修改：
themes\next\source\css_common\components\post\post-expand.styl

只能通过这里修改，如果直接用属性修饰img无效。
